# Student Management System (Python)

This is a command-line based Student Management System built using Python.  
It allows users to manage student records efficiently with features like adding, updating, deleting, and analyzing student data.

---

## Features

- Add new student
- View all students
- Search student by ID
- Update student details
- Delete student
- Calculate student average and grade
- View top N students based on performance
- View class statistics (total students, class average)

---

## Concepts Used

- Functions (modular programming)
- Lists and Dictionaries
- Input validation
- Looping and conditional logic
- Basic data analysis (averages, ranking)

---
